How to create a list

ol -> used for ordered list items
li-> used to write the list items.

<ol>
  <li>Mercury</li>
  <li>Venus</li>
  <li>Earth</li>
  <li>Mars</li>
</ol>

Output

  1. Mercury
  2. Venus
  3. Earth
  4. Mars

How to change list-style-type

list-style-type type is an attribute that changes the type list style like :->

<ol style="list-style-type : upper-alpha;" >
  <li>Mercury</li>
  <li>Venus</li>
  <li>Earth</li>
  <li>Mars</li>
</ol>

Output

  1. Mercury
  2. Venus
  3. Earth
  4. Mars
